# Data Files

All data files live in `~/.dotfiles/migrate/data/`.

## Brewfile

Homebrew bundle file. Managed manually (not auto-generated).

**Sections:**
- `brew "..."` — CLI tools (git, neovim, fzf, starship, zoxide, etc.)
- `cask "..."` — GUI apps (1password, ghostty, visual-studio-code, etc.)

**Updating:** Edit this file by hand. Do not regenerate it with `brew bundle dump`; it intentionally excludes packages you do not want.

## mas-apps.txt

App Store apps installed via `mas`.

```
# Format: id  name
1630034110  Bob
1579200037  Medis
1594061654  Navicat Premium
497799835   Xcode
```

**Updating:**
Edit this file by hand. Keep only App Store apps that should be installed on the new Mac.

## rsync-apps.txt

Apps that can't be installed via brew/mas — rsync'd as `.app` bundles.

```
# Format: app name (without .app suffix)
Discord
Telegram
WeChat
...
```

**Updating:** Add/remove app names. Each name must match `/Applications/<name>.app`.

## dock-layout.json

Dock icon layout exported from `com.apple.dock` plist.

```json
{
  "apps": [
    {"label": "Safari", "path": "/System/Volumes/Preboot/.../Safari.app"},
    {"label": "Google Chrome", "path": "/Applications/Google Chrome.app"}
  ],
  "others": [
    {"label": "Downloads", "path": "$HOME/Downloads"}
  ]
}
```

**Updating:**
```bash
/usr/bin/python3 -c "
import plistlib, json
with open('$HOME/Library/Preferences/com.apple.dock.plist', 'rb') as f:
    dock = plistlib.load(f)
apps = [{'label': i['tile-data']['file-label'],
         'path': i['tile-data']['file-data']['_CFURLString'].replace('file://','')}
        for i in dock.get('persistent-apps', [])]
others = [{'label': i['tile-data']['file-label'],
           'path': i['tile-data']['file-data']['_CFURLString'].replace('file://','')}
          for i in dock.get('persistent-others', [])]
print(json.dumps({'apps': apps, 'others': others}, indent=2, ensure_ascii=False))
" > data/dock-layout.json
```

## notification-prefs.plist

macOS Notification Center preferences copied from `~/Library/Preferences/com.apple.ncprefs.plist`.

**Behavior:**
- Restored by step 15 after app installation.
- Preserves source-machine allow/deny choices for bundle IDs already present in the plist.
- Filters `/Applications/*.app` entries to apps that exist on the target, avoiding stale notification rows for apps not installed on the new Mac.
- Does not globally pre-authorize unknown future apps; macOS can still prompt for apps that never appeared in the source preferences.

**Updating:**
Normally automatic via `./migrate.sh <target>`. To refresh manually:

```bash
cp ~/Library/Preferences/com.apple.ncprefs.plist data/notification-prefs.plist
```

## gpg/

Exported GPG key material and ownertrust. This directory is generated by `migrate.sh`, transferred to the target, and ignored by git because `secret.asc` contains private key material.

**Files:**
- `public.asc` — public keys exported with `gpg --armor --export`
- `secret.asc` — secret keys exported with `gpg --armor --export-secret-keys`
- `ownertrust.txt` — trust assignments exported with `gpg --export-ownertrust`

**Restoring:**
Handled by step 13:

```bash
bash ~/migrate/setup.sh 13
```

## npm-global.txt / pnpm-global.txt

Reference lists of global packages (not auto-installed).

**Updating:**
```bash
npm list -g --depth=0 | tail -n +2 > data/npm-global.txt
pnpm list -g --depth=0 | tail -n +2 > data/pnpm-global.txt
```

## vscode-extensions.txt

VS Code extension IDs, one per line.

**Updating:**
```bash
code --list-extensions > data/vscode-extensions.txt
```
